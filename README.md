# TASKFLOW
GESTOR DE TAREAS,

Aplicación web para crear y gestionar tareas.

TaskFlow permite agregar tareas, asignarles una prioridad, cambiar su estado (pendiente, en proceso o completada) y eliminarlas.
Las tareas se guardan en el navegador usando LocalStorage.

###Tecnologías usadas###

HTML

CSS

JavaScript

###Funcionalidades###

Crear tareas con título, descripción y prioridad

Cambiar el estado de una tarea

Eliminar tareas

Modo oscuro

Guardado automático en el navegador

###Cómo usar el proyecto###

Descarga o clona el repositorio

Abre el archivo index.html en tu navegador

Empieza a crear tareas

Web application for creating and managing tasks.

TaskFlow allows you to add tasks, assign them a priority, change their status (pending, in progress, or completed), and delete them.
Tasks are saved in the browser using LocalStorage.

###Technologies Used###

HTML

CSS

JavaScript

###Features###

Create tasks with a title, description, and priority

Change the status of a task

Delete tasks

Dark mode

Automatic saving in the browser

###How to Use the Project###

Download or clone the repository

Open the index.html file in your browser

Start creating tasks
